Short message    :	Attribute 'spatialSizeUnits' must not be set if 'hasOnlySubstanceUnits'='true'
Full message     :	If a <species> definition sets 'hasOnlySubstanceUnits' to 'true', then it must not have a value for 'spatialSizeUnits'.
Reference: L2V2 Section 4.8.5
 The <species> with id 's' should not have a 'spatialSizeUnits' attribute OR should have a 'hasOnlySubstanceUnits' attribute that is not set to 'true'.
