Short message    :	Layout attributes allowed on <referenceGlyph>.
Full message     :	A <referenceGlyph> object must have the required attributes 'layout:id' and 'layout:glyph' and may have the optional attributes 'layout:metaidRef', 'layout:reference' and 'layout:role'.  No other attributes from the Layout namespace are permitted on a <referenceGlyph>. 
Reference: L3V1 Layout V1 Section 3.11.1
  Attribute 'width' is not part of the definition of an SBML Level 3 Version 1 Package layout Version 1 <referenceGlyph> element.
