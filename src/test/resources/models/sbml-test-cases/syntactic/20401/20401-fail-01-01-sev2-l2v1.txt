Short message    :	Invalid 'id' attribute value for a UnitDefinition object
Full message     :	The value of the 'id' attribute in a <unitDefinition> must be of type 'UnitSId' and not be identical to any unit predefined in SBML. 
Reference: L2V1 erratum 14
 That is, the identifier must not be the same as any of the following predefined units: 'ampere' 'Celsius' 'gram' 'katal' 'metre' 'second' 'watt' 'becquerel' 'gray' 'kelvin' 'mole' 'siemens' 'weber' 'candela' 'henry' 'kilogram' 'newton' 'sievert' 'coulomb' 'hertz' 'litre' 'ohm' 'steradian' 'dimensionless' 'item' 'lumen' 'pascal' 'tesla' 'farad' 'joule' 'lux' 'radian' 'volt'. A <unitDefinition> with the id 'litre' is not allowed.
