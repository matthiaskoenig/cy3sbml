Short message    :	Attribute 'spatialSizeUnits' must not be set if the compartment is zero-dimensional
Full message     :	A <species> definition must not set 'spatialSizeUnits' if the <compartment> in which it is located has a 'spatialDimensions' value of '0'.
Reference: L2V2 Section 4.8.5
 The <species> with id 's' is located in 0-D <compartment> 'c' and therefore should not have a 'spatialSizeUnits' attribute.
