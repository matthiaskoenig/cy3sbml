Short message    :	Layout attributes allowed on <point>.
Full message     :	A <point> object must have the required attributes 'layout:x' and 'layout:y' and may have the optional attributes 'layout:id', and 'layout:z'.  No other attributes from the Layout namespace are permitted on a <point>.

  Attribute 'depth' is not part of the definition of an SBML Level 3 Version 1 Package layout Version 1 <position> element.
