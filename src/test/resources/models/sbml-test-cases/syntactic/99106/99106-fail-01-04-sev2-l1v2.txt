Short message    :	Invalid ordering of rules
Full message     :	The math field of an assignment rule structure can contain any identifier in a MathML <ci> element except for the following: (a) identifiers for which there exists a subsequent assignment rule, and (b) the identifier for which the rule is defined. (L2V1 Section 4.8.4).
 The AssignmentRule with variable 'p' refers to the variable 'z' within the math formula 'z'. 'z' is the subject of a later assignment rule.
