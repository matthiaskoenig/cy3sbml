Short message    :	Layout attributes allowed on <dimensions>.
Full message     :	A <dimensions> object must have the required attributes 'layout:width' and 'layout:height' and may have the optional attributes 'layout:id', and 'layout:depth'.  No other attributes from the Layout namespace are permitted on a <dimensions>. 
Reference: L3V1 Layout V1 Section 3.4.2
  Attribute 'metaidRef' is not part of the definition of an SBML Level 3 Version 1 Package layout Version 1 <dimensions> element.
