Short message    :	An empty list of Unit objects is not permitted in a UnitDefinition object
Full message     :	An SBML XML document must conform to the XML Schema for the corresponding SBML Level, Version and Release. The XML Schema for SBML defines the basic SBML object structure, the data types used by those objects, and the order in which the objects may appear in an SBML document. The <listOfUnits> container in a <unitDefinition> cannot be empty.
