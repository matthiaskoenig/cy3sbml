Short message    :	Invalid value for a Species 'units' attribute
Full message     :	The value of a <species>'s 'units' attribute is restricted. 
Reference: L2V5 Section 4.8.5
 The value of a <species>'s 'substanceUnits' attribute can only be one of the following: 'substance', 'mole', 'item', 'gram', 'kilogram', 'dimensionless', or the identifier of a <unitDefinition> derived from 'mole' (with an 'exponent' of '1'), 'item' (with an 'exponent' of '1'), 'gram' (with an 'exponent' of '1'), 'kilogram' (with an 'exponent' of '1'), or 'dimensionless'.  The current value ('time') is not allowed.
