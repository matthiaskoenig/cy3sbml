The model contains one compartment called C. There are three species called S1, S2 and S3 and two parameters called k1 and k2. The model contains one reaction defined as:

Reaction Rate
S1 → S2 C · k2 · S1

The model contains one rule which assigns value to species S3:

Type Variable Formula
Assignment S3 k1 · S2
