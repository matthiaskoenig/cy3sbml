The model contains three varying parameters called P1, P2 and P3 and one constant parameter called k1.

The model contains three rules:

Type Variable Formula
Rate P3 k1 · s
Rate P1 −(P3 · P1)
Rate P2 P3 · P1

where the symbol 's' denotes the current simulation time.

The initial conditions are as follows:

Value Units
Value of parameter P1 1.5 × 10−3 any
Value of parameter P2 0 same as P1
Value of parameter P3 1 × 10−3 second-1
Value of parameter k1 0.5 second-3
