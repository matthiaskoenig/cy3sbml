The model contains one varying compartment called C. There are three species called S1, S2 and S3 and four parameters called k1, k2, p1 and p2. The model contains two reactions defined as:

Reaction Rate
S1 + S2 → S3 k1 · S1 · S2 · C
S3 → S1 + S2 k2 · S3 · C
